# Contributing

Thanks for your interest in contributing.

## Development Setup

1. Create virtual environment.
2. Install dependencies from `requirements.txt`.
3. Run `uvicorn main:app --reload`.

## Pull Request Checklist

- Keep changes scoped and well documented.
- Add or update tests for changed behavior.
- Confirm no sensitive data is committed.
- Update README when UX/API changes.

## Commit Style

Use clear, action-oriented messages. Example:

- `feat(viewer): add plain/rich text document preview`
- `fix(upload): enforce file size and content-type limits`
